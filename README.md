
# MernStack E-commerce

Welcome to our MERN stack e-commerce site! This comprehensive web application leverages the power of the MERN stack — MongoDB, Express.js, React.js, and Node.js — to deliver a seamless online shopping experience. With the integration of Stripe, a leading payment gateway, users can securely complete their purchases, making this platform both user-friendly and secure.


Key Features - 

#User Authentication -
Enjoy a secure and personalized experience with our user authentication system. JWT (JSON Web Tokens) is employed to ensure the confidentiality and integrity of user information.

#Product Catalog -
Explore a diverse range of products showcased in an intuitive and visually appealing catalog. Each product comes with detailed information to help you make informed purchasing decisions.

#Shopping Cart - 
Effortlessly add and remove items from your shopping cart, providing a smooth and convenient shopping experience. The cart dynamically updates to reflect your selected items.

#Checkout Process -
Our checkout process is streamlined and efficient. With Stripe integration, you can confidently complete your purchase with various payment methods, ensuring a secure and reliable transaction.

#Order Management-
Keep track of your orders with our user-friendly order management system. View order details and stay informed about the status of your purchases.

Technologies Utilized

#Frontend -
React.js: A powerful JavaScript library for building interactive user interfaces.

Redux: State management for React applications, ensuring a predictable state container.

React Router: Facilitates navigation within the single-page application.

#Backend - 
Node.js with Express.js: A robust backend framework for building scalable and efficient server-side applications.

MongoDB: A NoSQL database for storing and retrieving product and user information.
Mongoose: An elegant MongoDB object modeling tool.

Payment Processing - 

 Stripe -: A secure and reliable payment gateway that allows users to make online transactions seamlessly.

Install my-project with npm

```bash

cd frontend 
npm Install
npm run dev

cd server
npm Install
npm start

```
