## Features

- **Responsive Design:** The website is designed to work seamlessly on various screen sizes, providing a consistent experience across devices.

- **User Authentication:** Secure user registration and login system to protect customer data and enable personalized experiences.

- **Dynamic Product Catalog:** The platform features a real-time updated product catalog with easy navigation and filtering options.

- **Efficient Checkout:** Streamlined checkout process with intuitive steps to reduce cart abandonment and improve user satisfaction.

- **Admin Dashboard:** An admin panel for easy management of products, orders, and site content.

## Tech Stack

- **Frontend:** React, tailwind css ,redux toolkit
- **Backend:** Node.js, Express
- **Database:** MongoDB (or your chosen database)
- **Authentication:** JWT (JSON Web Tokens)
- **Deployment:** vercel
